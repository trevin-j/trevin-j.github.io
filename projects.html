Projects

Below is a list of some of my best projects that I have worked on, or am currently working on. Many of these projects are very much a work in progress, and I work on them whenever I get the chance to. Not all of my public projects are on this list, but you can find them on my GitHub page. I also have several projects that I am working on that are not yet public.

FudgeRunner

FudgeRunner is an interpreter for the Brainfudge esoteric programming language. The interpreter was written in Java and therefore requires you to have Java installed to compile and run it. Included in the interpreter is the ability to interpret BF files and a fully functioning Brainfudge REPL. FudgeRunner runs in the command-line, and requires command line arguments to run.

The repository for this project can be found here, which contains instructions for building and using the program.

FudgeRunner++

FudgeRunner++ is a Brainfudge interpreter and compiler written in C++. FudgeRunner++ supports live interpreting files, compiling files, and command-line REPL. Similarly to FudgeRunner, FudgeRunner++ runs in the command-line, and requires command line arguments to run. Instructions for usage can be found in the repository linked below. Compiling files is done by first compiling a BF file to a C file, then using GCC to compile the C file to an executable. Because of this, the executable is fully standalone and requires nothing else to run. Unfortunately, FR++ is untested on Mac and Linux currently, but Windows 10 and 11 are supported.

Check out the repository for this project here.

Sorting Visualizer

Sorting Visualizer is a fun little program written in C++, and using SFML, designed to visualize different sorting algorithms. While I haven't spent much time since I initially started this project, there are still some changes that would benefit this program. The program uses polymorphism to make it easier to add new sorting algorithms.

Instructions for usage can be found in the repository here.

HTML Builder Library

HTML Builder is a C++ header-only library used for the generation of HTML. The repository contains good documentation for usage, and the library is quite small and easy to use. I initially created this library for me to use in my own markdown parser, but the parser is far from finished and is not public.

Check out the repository for this project here.

Data Structures Tutorial

This project consists of 3 short tutorials designed to teach you about stacks, linked lists, and trees.

You can view the live site here, or view the repository here.

EZTTS

EZTTS is a Python package for easily generating text to speech. The package is designed to be highly modular and supports multiple TTS APIs. With the wide variety of TTS tools out there, it can be hard to pick one to learn and use, so EZTTS takes several and puts them together with only the need to learn a single interface.

Check out the repo here.
